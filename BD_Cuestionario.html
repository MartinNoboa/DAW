Bases de Datos : Cuestionario

¿Qué es entonces una base de datos - DB?

Una base de datos es el conjunto de tablas interrelacionadas.


¿En qué casos es conveniente usar bases de datos?

Es conveniente usar una base de datos cuando se maneja gran cantidad de datos, y se busca poder actualizarlos y leerlos de manera rapida y eficiente.


¿Qué es un sistema de gestión de base de datos - DBMS?

Un sistema de gestion de base de datos es un programa que permite interactuar con la base de datos de manera sencilla y optima.


Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • Simplificar la interaccion entre el usuario y la base de datos.
  • Optimizar los procesos en la base de datos
  • Reducir tiempo de consultas
