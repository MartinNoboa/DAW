¿Cuáles son los valores estándar para la propiedad visibility?

  • Visible
  • Hidden
  • Collapse(tables)
  • Inherit

¿Qué es el zIndex y para qué sirve?

La propiedad CSS z-index indica el z-order de un elemento y sus descendientes. Cuando varios elementos se superponen, z-order determina cuales cubren a otros.
