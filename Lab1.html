Preguntas

¿Cuál es la diferencia entre Internet y la World Wide Web?

El internet es el conjunto de redes, mientras que la World Wide Web es un medio de acceso al Internet. La WWW funciona a traves de documentos de hipertexto(El hipertexto es una estructura no secuencial que permite crear, agregar, enlazar y compartir información de diversas fuentes por medio de enlaces asociativos y redes sociales)

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET : usado para pedir data de una fuente especifica.

HEAD : es igual a GET, solo no retorna informacion.

POST : envia datos al servidor para crear o actualizar un recurso.

PUT : envia datos al servidor para crear o actualizar un recurso.

La diferencia entre PUT y POST es que PUT es idempotente, es decir, a partir de una aplicacion inicial, el resultado no cambia.

DELETE : elimina un recurso esoecifico.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Cuando se manda el formulario se usa el metodo GET. Cuando se responde el formulario, se usa el metodo POST.
